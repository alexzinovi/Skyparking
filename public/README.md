# Public Assets Folder

Place your static assets here (images, logos, fonts, etc.)

## How to use:
1. Add your files to this folder (e.g., `logo-header.png`)
2. Reference them in your code with `/filename.png`
3. Example: `<img src="/logo-header.png" alt="Logo" />`

## Current assets needed:
